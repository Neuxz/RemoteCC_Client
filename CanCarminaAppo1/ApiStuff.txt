https://cancarmina.de/?m=api&doku=1

DOKUMENTATION der API für Chorsystem https://cancarmina.de/?m=api&doku=1
   für alle Requests gilt:
   CODE    = persönlicher Mitglieder Code
   cancarmina.de kann auch durch andere Domains ausgetauscht werden (z.Zt. nur: 4laut.de)

Textausgabe zum Login
   http://cancarmina.de/?m=api&c=CODE
   Output:   true

Terminliste ausliefern
   http://cancarmina.de/?m=api&c=CODE&t=alle
   t       = 'alle'
   Output:   Ausgabe der Terminliste ab minus 7 Tage. Format: JSON, mit wichtigen Infos der Termintabelle

Mitgliederinfos ausgeben
   http://cancarmina.de/?m=api&c=CODE&mgl=XXX
   XXX     = 'foto' Mitgliederfoto (klein)
             'foto-gross' Mitgliederfoto (groß)
             'id'   Mitglieds ID (Integer)
   Output:   s.o. Klartext, bzw. Foto

Termindetails ausliefern
   http://cancarmina.de/?m=api&c=CODE&t=TT
   TT      = ID des Termins, Bezeichnung in der Datenbank: trm_id
   Output:   Ausgabe der Termindetails (komplett) mit Anmeldestatus des Mitglieds

Termin an/abmelden
   http://cancarmina.de/?m=api&c=CODE&t=TT&a=AA
   TT      = ID des Termins, Bezeichnung in der Datenbank: trm_id
   AA      = 'n' für anmelden
             'b' für abmelden
   Output:   Anzeige in Klartext, welche Aktion durchgeführt wurde

Alle offenen Termine anmelden
   http://cancarmina.de/?m=api&c=CODE&a=alleanmelden
   Output:   Anzeige in Klartext

Umfrage mit JA / NEIN abstimmen
   http://cancarmina.de/?m=api&c=CODE&u=UU&a=AA
   UU      = ID der Umfrage, Bezeichnung in der Datenbank: umf_id
   AA      = 'j' für ja
             'n' für nein
   Output:   Anzeige in Klartext, welche Aktion durchgeführt wurde

Forumsbeitrag ausgeben / erstellen
   http://cancarmina.de/?m=api&c=CODE&frm=NN
   NN      = Wenn Zahl: Anzahl, wieviele Forumsbeiträge ausgegeben werden sollen
   NN      = Wenn Text, wird der Beitrag ins Forum eingetragen
   Output:   Ausgabe der NN letzten Forumsbeiträge als JSON

Error Codes
   367     = CODE falsch: Mitglied nicht gefunden. Webmaster wird per E-Mail benachrichtigt.
